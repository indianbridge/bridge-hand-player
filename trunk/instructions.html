General Instructions

  • Start with a clean slate to enter the hands graphically. After entering a full deal ( 13 cards to everyone ) you will be taken to auction area.
  • Or provide the hands in the URL using handviewer format and proceed directly to auction area. After you have entered a valid auction you will be taken to play area.
  • Or provide hands and auction and proceed directly to play area. You can create a line of play by clicking on cards.
  • At any point you can create a BBO handviewer url or BW embed code which will include everything setup so far.
  • Or you can provide hands, auction and play and proceed to viewer/player area. You can either follow the play using control buttons in the bottom or choose to click on cards at any point in the play to branch out to form your own line of play to test the outcome.
  • You can change things like dealer, player names, board number, vulnerability as well as add annotations to bids and plays by clicking in appropriate places.
  • You should be able to take any handviewer url that includes 4 full hands and plug it in here. If you don't have 4 full hands then you will be shown the hand screen to specify the full hand.

Additional Features

  • You can specify more than one line of play by adding more p parameters in the url. Each additional p parameter should have a name immediately following like this example.
  • You can load any of the lines of play using the Load Play button in the bottom.
  • You can also create your own line of play and and save it using the Save Play button.
  • Multiple lines of play can only be exported to the BHP (this tool) extended format URL. When you export to BBO or BW only the currently active play will be included since that format only supports one line if play.
  • You can provide a direct link to a specific point in the play by adding a #play_number at the end of the URL where play_number will be replaced by the actual point in the play you want a direct link to. Each card played increments the play number by 1. So 4 indicated the end of trick 1, 10 indicates 2 cards played to trick 3 etc. If you are loading a deal for the first time the play number will refer to the first listed line of play. If a deal has already been loaded and you change the play number then it will refer to the currently active line of play. For example this link will load hands, auction and play and take you directly to end of trick 3 (play number 12).
  • A t parameter can be used to specify a title and general description for the entire deal like this example. This will displayed in the top left area.

Create Hand Instructions

  • The left pane contains unassigned cards. Assigned cards will be face down while unassigned cards will be face up.
  • The right pane shows the currently active hand. Click on another hand to make that active.
  • Click on unassigned cards in the left pane to assign it to the currently active hand.
  • Click on any card in left pane to remove it from the active hand and add it to the unassigned cards.
  • Click Assign Rest Automatically to assign the remaining unassigned cards to hands with space available. Note that this is not a random assignment. This button is useful when three hands have been assigned and you want to assign the remaining cards to the fourth hand.
  • Click Unassign All Cards to move all assigned cards to unassigned status and start fresh.
  • When the hands form a valid deal (13 cards each) the Accept Hands button will be enabled and can be clicked to go to the next area.
  • At any point you can use the Generate URL (top) button to generate BBO or BW or BHP url/embed code that would include all of the currently saved information

Create Auction Instructions

  • The left pane contains the dealer and the bidding box. Bids that are not valid in the current context will be hidden.
  • The right pane contains the auction so far.
  • Click on the directions next to Dealer to change the dealer. Note that if you have a developing auction then the entire auction will be wiped out ( you will be asked to confirm change of dealer if an auction exists )
  • Click on any bid in the bidding box to add it to the current position in the auction.
  • Click on Undo Last Bid to remove the last bud in the auction.
  • Click on any bid in the auction (right pane) to add an explanation/description.
  • Click on Edit Hands button (bottom) to go back to editting hands. The current auction will be stored to be restored when you return to the auction area.
  • Click on Add All Pass button (bottom) to automatically fill in passes to complete the auction.
  • Click on Undo All Bids button (bottom) to clear out all bids and start from scratch.
  • When a valid auction has been created ( ending in 3 passes ), the Accept Auction button (bottom) will be enabled and can be clicked to proceed to the next area
  • At any point you can use the Generate URL (top) button to generate BBO or BW or BHP url/embed code that would include all of the currently saved information

Create Play Instructions

  • If a play has been specified in the URL then you can use the undo and redo play control buttons (bottom) to step through the plays. You can also add a #n to the end of the URL where n is the numeric index of the play you want to move to.
  • At any point while following the play or if no play has been specified you can start creating your own line of play by clicking on cards. Playable cards will be highlighted in green outline.
  • Clicking on a played card ( cards shown on the table ) lets you enter a play annotation.
  • You can also add an annotation to the auction by clicking on any bid.
  • Plays can be saved and loaded using the Save play and Load play buttons (bottom)
  • You can go back and edit the hands or the auction using the Edit Hands and Edit Auction buttons (bottom)
  • Click on player names to change the names, click on Board to change the board number, click on compass to change the vulnerability.
  • At any point you can use the Generate URL (top) button to generate BBO or BW or BHP url/embed code that would include all of the currently saved information. Note that for BBO and BW only the currently active play will be included.
