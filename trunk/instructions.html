Bridge Hand Player (BHP) Instructions

  1. The Bridge Hand Player (BHP) Tool tries to extend the functionality of the Bridge Base Online's Hand Viewer tool . The key additions are ability to visually play out different lines, add a line of play to the handviewer url, save multiple lines of play to be shared with others.
  2. BHP hands are specified using the same format as the BBO Handviewer i.e., parameters are appended to the URL. BHP supports all but the following parameters of the BBO Hand Viewer : c - number of tricks claimed, k -kibitz, lin - Contents of actual .lin file to display, linurl - URL of external .lin file in XML wrapper, myhand - Identifier of deal stored in myhands. For details of all the parameters go to the hvdoc page.
  3. BHP supports one additional parameter and extends the p (line of play) parameter. A t parameter can be used to add some text (maybe a title and an author or some other general commentary) that is shown at all times. HTML is supported, so for example "t=<h4>Relay Double</h4> by Kit Woolsey" can be used to add a title and author. BHP also supports specification of multiple lines of play. While the HandViewer p parameter can be specified as usual additional lines of play can be specified by using the parameter p and the name of the play immediately after and the play specified in the usual way. For example "pLine1=c2c3cacqhah3h2hk&pLine2=c2c3cacqc5ckc7c4" can be used to specify 2 alternate lines of play.
  4. Once the hand and auction and play has been loaded, the play can be stepped through using the buttons on the bottom just like the hand viewer. However at any point you can explore your own line of play by clicking on playable cards (highlighted by green outline). The original lines of play can be restored at any time by using the Line drop down at the top menu bar. When you have explored a different line of play, that line can be saved and then you can generate a new url ( see next step) with the new line appended.
  5. At any point, you can generate a BBO hand viewer url to share with others or embed in some webpage. Note that this will only include the currently active line of play. A Bridge Winners embed code can also be generated to be included when posting articles on the Bridge Winners Page. A BHP url can also be generated which will include all saved lines of play as well the current line of play (if it is not saved).
  6. You can link directly to any point in the play by adding # followed by a number indicating the play. For example adding #8 at the end would link to the end of trick 2 directly.
