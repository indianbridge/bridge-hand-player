Bridge Hand Viewer/Player Instructions

  1. This tool lets you play out bridge hands in double dummy fashion to experiment the outcome of different lines of play.
  2. Hands are specified using the format of BridgeBase Hand Viewer. Instructions on that format and what parameters can be specified with the url can be found at http://www.bridgebase.com/tools/hvdoc.html. While the hand viewer supports several parameters listed at the end of the above link, this tool does not support them all. The following parameters are not supported currently :
    1. c number of tricks claimed (This is not relevant given the purpose of this tool)
    2. k kibitz (This is not relevant given the purpose of this tool)
    3. lin Contents of actual .lin file to display (need to implement parsing of lin contents)
    4. linurl URL of external .lin file in XML wrapper (need to implement parsing of lin contents)
    5. myhand Identifier of deal stored in myhands (need to implement parsing of lin contents)
  3. After a hand has loaded you can click on cards to play ( cards that can be played are highlighted with a green outline ). The undo and redo buttons can be used to take back and redo plays.
